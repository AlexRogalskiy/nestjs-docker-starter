## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository with Docker integrated.

## Requirements

- Docker

## Installation

```bash
$ npm install
```

## Running the app

```bash
docker-compose up
```

This runs the app in debug mode. You should be able to use the VSCode debugger (see `.vscode/launch.json`).

### App commands

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
